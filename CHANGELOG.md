# Changelog

## 0.2.1 - 2026-07-23

- Fixed GIF media probing so animated GIFs open as timed media instead of still images.
- Restored GIF playback in the desktop UI through the existing animated preview path.
- Restored exporting GIF inputs to MP4, MOV, GIF, and PNG sequence outputs from the export dialog.
- Added a regression test covering GIF probe behavior.

## 0.2.0

- Added crop, trim, resize, rotation, flip, preview, and export workflows for desktop media editing.
- Added timeline thumbnails, exact trim entry, playback speed control, and headless export support.
